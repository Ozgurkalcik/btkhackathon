# Finance Expert 🚀

Finance Expert, kişisel finans yönetimini bir sonraki seviyeye taşıyan premium, yapay zeka destekli bir akıllı finans asistanıdır. Sıradan bütçe takip uygulamalarının aksine, harcamalarınızı sadece kaydetmekle kalmaz; yaşam tarzınızı analiz eder, alışkanlıklarınızı öğrenir ve size 20 yıllık kıdemli bir servet yöneticisi gibi finansal içgörüler sunar.

## ✨ Temel Özellikler

### 1. Zenginleştirilmiş Veri Boru Hattı (Enriched Data Pipeline)
- **Açık Bankacılık Uyumlu:** Tüm banka hesaplarınızı tek bir ekranda (Konsolide Bakiye) toplar.
- **Yapay Zeka (Gemini API) Entegrasyonu:** Ham banka/POS dökümlerini okur, temizler ve anlamlı kategorilere (Örn: "NIKBEY GIDA" -> "Market / Malatya") dönüştürür.

### 2. Davranışsal Analitik Motoru (Analytics Engine)
- **Sağlık & Yaşam Tarzı Analizi:** Sadece "ne kadar" harcadığınızla ilgilenmez. McDonald's, Burger King, Starbucks gibi markalarda ayda kaç kere harcama yaptığınızı sayar ve sağlıksız beslenme sınırını aştığınızda uyarı verir.
- **Gece Kuşu Uyarısı:** Dürtüsel gece alışverişlerini tespit eder ve bütçe planlamanızı korur.
- **Mikro Sızıntılar (Latte Faktörü):** Küçük ama çok sık yapılan harcamaları yakalar ve cüzdanınızdaki sızıntıyı engeller.

### 3. Premium Glassmorphism Arayüz (UI/UX)
- B2B ve sıkıcı "şirket" tasarımları yerine, kullanıcıya "Özel Bankacılık" hissi veren premium koyu mod (Deep Dark Mode) ve cam efektli (Glassmorphism) kart tasarımları kullanılır.
- Dinamik harcama çubukları ve veri görselleştirme altyapısıyla karmaşık finansal verileri okunması çok kolay özetlere dönüştürür.

## 🛠 Kullanılan Teknolojiler

- **Flutter / Dart:** Çapraz platform, yüksek performanslı mobil uygulama çerçevesi.
- **Yapay Zeka (LLM):** İşlem ayrıştırma ve akıllı çıkarımlar için.
- **Özel Bileşenler:** CustomPaint ile performanslı harcama grafikleri ve `FittedBox` / `GlassContainer` destekli akıcı UI.

## 🚀 Başlangıç

Bu projeyi yerel ortamınızda çalıştırmak için:

1. Depoyu klonlayın:
   ```bash
   git clone <repo-url>
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   flutter pub get
   ```
3. Uygulamayı başlatın (Varsayılan olarak karanlık tema aktiftir):
   ```bash
   flutter run
   ```
